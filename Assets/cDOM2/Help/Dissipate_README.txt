Installation
------------
Unzip cDOM-DMP-Dissipate.
Store the *.unr or *.ut2 file in your Maps\ directory along with the others 
(see guide below), then either doubleclick on it, or type open Dissipate in 
the console, or select the Map in a Botmatch or an Internet game.

Guide:
*.unr and *.ut2 files go in Maps\
*.utx files go in Textures\
*.umx and *.ogg files go in Music\
*.uax files go in Sounds\
*.u, *.int, and *.ini files go in System\
*.usx files go in staticmeshes\
*.ukx files go in animations\
*.ka  files go in karma\
*.upx files go in prefabs\

Extended Description
--------------------

Partially imbedded in an off coast island near the RobotFactory venue this base provides you all the peace and quiet you need to dominate your opponent. Specially constructed by Liandri for the revival of the classic domination gametype.

Author's Notes
--------------

Building this one was a real bitch. A2597 first built his version of the map, with lots of unaligned stuff making the topdown view look like a large plate of green, yellow and blue spaghetti. When he started having serious lighting/FPS problems he eventually turned to me. I was interested by the layout but because of the 'spaghetti' I decided to build the map again from scratch. It turned out quite well, all HOM's are hidden, lighting is okay, stuff is aligned and it offers some very nice gameplay. Thanks to Dregs for being there when needed.
I felt little for using the same old RobotFactory skybox copy/paste, so instead I made it look slightly dusky. The imbedded-in-an-island-feeling is created through use of lots of rock/plant/tree/foliage meshes and some terrain which could be seen through those windows. I accidentically created an in my vision good looking visual, namely the pipes in the lower section of the map, visible through the glass floor. In the middle the meshes cut off at a certain point, but since the sunlight drops down straight it almost seems if the pipes are partially lit by the sunlight.
The problem with this map was the lightmapping. At a certain point the editor would make various surfaces with identical lightmaps look like one has lightmap set to 1 and the other to 256. This was one of the reasons to start anew. Other reasons were whacky brush/mesh placement and in my view pretty bad lighting. So I started this map from scratch, made some extra meshes in the mylevel.additional and mylevel.Sjosz folders to fill up any unfitting stuff. On behalf of both A2597 and myself have fun and give feedback!
(and curse me for forgetting about the bShadowCast option of the window meshes!)
